üî¨ Working with Weather Data

Our project focuses on analyzing weather data using NetCDF files, a standard format in climate and meteorological sciences.

About This Project

The MISpace Hackathon project focuses on weather data science and machine learning. We work with:

  • NetCDF Files: Industry-standard format for weather and climate data
  • Weather Datasets: Temperature, precipitation, wind speed, and atmospheric pressure
  • Machine Learning: Predictive models for weather forecasting and pattern recognition
  • Data Visualization: Interactive charts and maps for weather data exploration

Technologies

Python NetCDF4 xarray Pandas Scikit-learn Matplotlib Jupyter

üíª Complete Example

# Complete workflow example
import xarray as xr
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
from sklearn.preprocessing import StandardScaler

# 1. Load NetCDF file
ds = xr.open_dataset('weather_data.nc')

# 2. Extract variables
temp = ds['temperature']
precip = ds['precipitation']

# 3. Convert to pandas DataFrame for ML
df = pd.DataFrame({
    'temperature': temp.values.flatten(),
    'precipitation': precip.values.flatten()
})

# 4. Clean data
df = df.dropna()

# 5. Feature scaling
scaler = StandardScaler()
df_scaled = scaler.fit_transform(df)

# 6. Create visualizations
plt.figure(figsize=(12, 6))
plt.subplot(1, 2, 1)
plt.plot(temp.mean(dim=['lat', 'lon']))
plt.title('Average Temperature Over Time')
plt.xlabel('Time')
plt.ylabel('Temperature (¬∞C)')

plt.subplot(1, 2, 2)
plt.plot(precip.mean(dim=['lat', 'lon']))
plt.title('Average Precipitation Over Time')
plt.xlabel('Time')
plt.ylabel('Precipitation (mm)')
plt.tight_layout()
plt.show()
